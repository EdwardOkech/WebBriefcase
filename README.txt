Web Briefcase is an application that enables web developers manage their applications details easily and effeciently,
the app has the following features:-

  1). Stores application login details securely i.e (cpanel username & passwords, CMS's admin details).
  2). Client details (phone,email,skype).
  3). Client management (sends notifications to clients when domain about to expire, send bulk messages)
  4). Subscribe to latest & trending news in web development industry.
  5). Browse jobs posted on your skills.
  6). Connect and share with fellow devs.
